# 項目名稱

HTTP APIs for Account and Password Management

該api提供了創建使用者以及驗證帳戶密碼的功能，主要基于Restful風格，接受JSON格式的輸入，並且回傳JSON格式的輸出。
該api使用Python編寫，使用到的套件為下列，請確保使用前環境滿足需求：
    Flask
    SQLAlchemy
